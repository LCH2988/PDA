🤖 關鍵字自動回覆
🔑 包含「捐款」或「捐」
自動回覆捐款方式說明，包含 LIFF 表單連結、快速捐款方式、指令捐款說明及匯款帳號。
例：請問如何捐款？
🔑 包含「帳號」或「匯款」
自動回覆完整匯款帳號資訊（土地銀行內壢分行）及線上表單連結。
例：請問匯款帳號是多少？
🔑 包含「謝謝」或「感謝」
自動回覆感謝訊息，傳遞祝福語。
例：謝謝大家的幫忙！

🔐 權限說明
🟢
所有人：無需設定，任何 LINE 用戶皆可使用
🟡
管理員（A2 欄位）：需在 #A2_權限設定 工作表的 A2:A10 欄位中登記 LINE User ID
可使用：代、改、改2、捐款芳名錄、活動公告、活動統計等指令
🔴
超級管理員（D 欄位）：需在 D2:D10 欄位中登記 LINE User ID
可使用：911、922、933、955、@大會統計、PD、PDA1 等進階指令
📌 指令格式說明

✅ 指令與參數之間請使用 「+」 號分隔

✅ 金額請輸入純數字，不需加「元」或逗號

✅ 快速捐款格式需以「元」結尾，如：我要隨喜樂捐1000元

✅ 修改指令的序號可從芳名錄的 [序號] 欄位查詢

⚠️ 指令前後請勿加入多餘空格，避免系統無法識別
