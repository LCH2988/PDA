🚀 安裝設定

📦第一次安裝

步驟 1：建立 Google Sheets

  1. 開啟 Google Sheets
  2. 建立新的空白試算表
  3. 將試算表命名為「帕金森氏症便秘改善研究」
  4. 記下試算表的 ID（網址中的長字串）

步驟 2：安裝程式碼

  1. 在試算表中點選「擴充功能」→「Apps Script」
  2. 刪除預設的程式碼
  3. 貼上完整的系統程式碼
  4. 修改 CONFIG 中的 SPREADSHEET_ID
  5. 點選「儲存」並為專案命名
const CONFIG = { SPREADSHEET_ID: '你的試算表ID', // 其他設定... };

步驟 3：初始化系統

  1. 執行 initializeSystemFixed() 函數
  2. 授權必要的權限（首次執行會要求授權）
  3. 等待系統建立所有工作表和設定
  4. 確認所有工作表都已正確建立

步驟 4：設定觸發器

  1. 在 Apps Script 中點選「觸發器」（時鐘圖示）
  2. 點選「新增觸發器」
  3. 設定編輯觸發器：
    • 函數：onDataChange
    • 事件來源：來自試算表
    • 事件類型：編輯時
  4. 儲存觸發器
✅ 安裝完成！ 系統會自動建立自定義選單「🏥 帕金森便秘研究系統」，重新整理頁面即可看到。

🚨 問題排除

❓常見問題 FAQ

❌ 系統選單沒有出現 +
解決方法：
  1. 重新整理試算表頁面（F5 或 Ctrl+R）
  2. 確認程式碼中包含 onOpen() 函數
  3. 重新執行 initializeSystemFixed()
  4. 檢查 Apps Script 權限是否正確授權
🔄 分析沒有自動執行 +
解決方法：
  1. 執行 checkTriggerStatus() 檢查觸發器
  2. 手動設定編輯觸發器（函數：onDataChange）
  3. 確認至少有 3 筆完整資料
  4. 檢查系統日誌是否有錯誤訊息
📊 統計結果異常 +
解決方法：
  1. 執行 checkDataQuality() 檢查資料品質
  2. 確認數值在合理範圍內（排便次數 0-10）
  3. 檢查是否有空白或錯誤資料
  4. 重新執行 manualAnalysis()
📧 郵件發送失敗 +
解決方法：
  1. 檢查郵件地址格式是否正確
  2. 確認 Gmail 發送額度未超限
  3. 重新授權 Apps Script 郵件權限
  4. 檢查垃圾郵件資料夾
💾 備份功能異常 +
解決方法：
  1. 檢查 Google Drive 儲存空間
  2. 確認 Drive API 權限
  3. 手動執行 backupData()
  4. 檢查備份資料夾權限
