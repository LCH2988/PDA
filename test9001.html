登入系統
註冊、電話登入、LINE LIFF 快速登入
📝

一般電話註冊

輸入姓名與電話號碼完成註冊。系統自動取電話後 9 碼作為個人 ID，適合無使用 LINE 的長輩。

電話後9碼=個人ID
💚

LINE LIFF 快速註冊

點擊「LINE 快速登入」，系統自動抓取 LINE ID 及 LINE 名稱。首次登入需實名制填寫姓名與電話（一次即可）。

LINE ID=個人ID
🔑

電話登入

輸入姓名與電話後 9 碼即可登入。不需記憶密碼，操作簡單，適合長輩使用。

免密碼登入
🟢

LIFF 一鍵登入

已有 LINE 帳號者，點擊「LINE 登入」一鍵進入系統。首次登入需完成實名制，之後直接進入主頁。

一鍵登入
1

開啟系統網址

從瀏覽器開啟 GitHub Pages 網址，或從 LINE 點擊 LIFF 連結進入。建議加入書籤或安裝至手機桌面。

2

選擇登入方式

  • 🟢 LINE 登入：點擊綠色按鈕，自動跳轉 LINE 授權
  • 📞 電話登入：輸入姓名＋電話後 9 碼
  • 📝 首次使用：點擊「立即註冊」
3

首次 LINE 登入（實名制）

系統偵測到新帳號，顯示實名制表單，填寫真實姓名與聯絡電話後完成綁定，之後登入不再需要填寫。

4

進入主頁

登入成功後顯示主頁，依照使用者角色權限顯示對應功能選單。

準備印章圖檔

使用掃描機或手機拍攝協會關防印章，存為 PNG 格式（透明背景），建議尺寸 300×300px 以上。

2

進入系統設定

3

上傳印章圖檔

點擊「選擇檔案」→ 選取 PNG 圖檔 → 點擊「上傳印章」，系統儲存至 Google Drive 並記錄 File ID。

4

驗證效果

任意開立一張測試收據，確認 PDF 右下角出現印章圖樣，即表示設定成功。

部署說明
從零開始上線的完整步驟
1

📋 建立 Google Sheets

  1. 登入 Google 帳號，開啟 Google Sheets
  2. 新增試算表，命名為「帕金森協會管理系統」
  3. 依序建立 10 個工作表（會員資料、繳費紀錄、捐款紀錄、收入明細、支出明細、會計科目、帳戶設定、預算設定、管理人員、系統日誌）
  4. 記錄試算表 ID（網址列 /d/ 後的字串）
2

⚙️ 部署 Google Apps Script

  1. 在試算表中點選「擴充功能」→「Apps Script」
  2. 將所有 .gs 程式碼貼入對應檔案
  3. 修改 Code.gs 第 5 行的 SHEET_ID
  4. 點選「部署」→「新增部署作業」→「網頁應用程式」
  5. 執行身分選「我」，存取權選「所有人」
  6. 複製部署後的 Web App URL
3

🌐 部署前端至 GitHub Pages

  1. 在 GitHub 建立新 Repository（公開）
  2. 上傳 index.html、manifest.json、service-worker.js、icons/ 資料夾
  3. 進入 Repository Settings → Pages → 選擇 main branch
  4. 等待約 1 分鐘，取得 https://帳號.github.io/repo名稱/ 網址
4

🔧 設定前端參數

  1. 開啟 index.html，找到第 10 行
  2. 將 API_URL 替換為步驟 2 的 Web App URL
  3. 將 LIFF_ID 替換為步驟 5 的 LIFF ID
  4. 儲存並推送至 GitHub
5

💚 設定 LINE LIFF

  1. 前往 developers.line.biz
  2. 建立 Provider → 新增 LINE Login Channel
  3. 切換至 LIFF 分頁 → Add LIFF app
  4. Size 選 Full，Endpoint URL 填入 GitHub Pages 網址
  5. Scope 勾選 profile、openid、email
  6. 複製 LIFF ID（格式：1234567890-AbCdEfGh）
6

🔔 設定自動觸發器

  1. 在 Apps Script 編輯器執行 setupTriggers() 函式一次
  2. 系統自動建立每日早上 9 點的預算檢查觸發器
  3. 確認「觸發器」頁面出現 dailyBudgetCheck
  4. 測試：手動執行 dailyBudgetCheck() 確認 Email 正常寄出
7

🖊️ 上傳協會印章

  1. 以理事長帳號登入系統
  2. 進入「系統設定」→「印章管理」
  3. 上傳 PNG 透明背景印章圖檔
  4. 測試開立一張收據，確認印章出現在 PDF 右下角
8

👔 建立管理人員

  1. 在 Google Sheets「管理人員」工作表手動填入第一位理事長資料
  2. 填寫：個人ID（LINE ID 或電話後9碼）、姓名、職稱英文（chairman）、任期起迄
  3. 以該帳號登入系統，確認可進入系統設定
  4. 再透過系統介面新增其他管理人員

❓
常見問題 FAQ
使用上常見的疑問與解答
❓LINE 登入後顯示「請重新登入」怎麼辦？▼
Session Token 有效期為 24 小時，超過後需重新登入。請點擊「LINE 登入」重新授權即可，資料不會遺失。若持續出現此問題，請確認手機時間設定是否正確。
❓收據 PDF 沒有印章，怎麼處理？▼
請確認理事長已在「系統設定 → 印章管理」上傳印章圖檔。印章必須為 PNG 格式、透明背景。若已上傳但仍無印章，請重新上傳一次，或聯絡系統管理員確認 Google Drive 儲存空間是否足夠。
❓繳費後沒有收到 Email 通知？▼
請確認：① 個人資料中已填寫正確的 Email 地址；② 查看垃圾郵件資料夾；③ 系統 Email 由 Google Apps Script 帳號發送，請將該地址加入白名單。若仍未收到，請聯絡行政秘書長確認繳費紀錄是否已登錄。
❓我是出納，為什麼無法新增支出？▼
新增支出需要會計（Lv.4）以上的權限，出納（Lv.3）只能登錄收入、繳費與捐款。若需要新增支出，請聯絡會計或總幹事操作，或由理事長調整您的職稱權限。
❓如何匯出會員名單給 Excel？▼
有兩種方式：① 在系統「報表功能」中下載 CSV 格式，用 Excel 開啟；② 直接開啟 Google Sheets 的「會員資料」工作表，點選「檔案 → 下載 → Microsoft Excel (.xlsx)」。建議使用方式 ② 取得最完整的資料。
❓系統可以同時多人使用嗎？▼
可以。Google Apps Script 支援多人同時存取，但請注意：同一時間大量請求可能受到 Google 的每分鐘呼叫次數限制（約 100 次/分鐘）。一般協會使用規模不會達到此上限。若遇到「服務暫時無法使用」，稍等 1-2 分鐘後重試即可。
❓資料會遺失嗎？如何備份？▼
Google Sheets 本身有版本歷史紀錄功能，可還原任意時間點的資料。建議額外設定每週自動備份：在 Apps Script 新增觸發器，每週一執行備份函式，將 Sheets 匯出為 Excel 並寄送至指定信箱。資料儲存於 Google 伺服器，安全性極高。
❓手機畫面字體太小，可以調整嗎？▼
系統已針對無障礙設計，基本字體為 16-20px。若仍覺得太小，可使用手機的「輔助使用 → 字體大小」功能放大全系統字體，或在瀏覽器設定中調整文字縮放比例。系統採用相對單位設計，放大後版面不會跑版。
❓如何更換新任理事長的帳號？▼
由現任理事長登入系統，進入「系統設定 → 管理人員設定」，將舊理事長的任期迄日設為今日，再新增新任理事長的個人ID與任期。系統會在下次登入時自動套用新的權限設定。新任理事長需重新上傳印章（若印章有更換）。
❓捐款收據可以用來報稅嗎？▼
可以。系統開立的捐款收據含有協會關防印章、收據編號、捐款金額、捐款人姓名等完整資訊，可作為綜合所得稅列舉扣除憑證。但請確認貴協會已取得財政部核准的捐贈扣除資格，並在收據上標示統一編號與核准文號。
