🔗 步驟一：部署 GAS Web App

  1. 開啟您的 Google 試算表 → 點選「擴充功能」→「Apps Script」
  2. 建立 4 個 .gs 檔案：Code.gs、DataReader.gs、DataWriter.gs、Helpers.gs
  3. 貼上對應程式碼，修改 CONFIG.SHEET_ID 為您的試算表 ID
  4. 點選「部署」→「新增部署」→ 類型選「網頁應用程式」
  5. 執行身份：我；存取權：任何人 → 點「部署」
  6. 複製產生的 Web App URL 貼到下方
