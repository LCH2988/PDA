GitHub 註冊與設定指南

從零開始，帶您完成帳號建立、安全防護與開發者環境配置

1. 帳號註冊 2. 安全與個人化 3. 開發者 SSH 設定 4. 建立首個倉庫
步驟一

註冊 GitHub 帳號

  1. 打開瀏覽器，前往 GitHub 官方網站 。
  2. 點擊網頁右上角的 「Sign up」（註冊）按鈕。
  3. 根據系統的對話式引導，依序輸入您的資訊：
    • Email：輸入您的電子信箱。
    • Password：設定高強度的密碼。
    • Username：設定您的使用者名稱（這會是您未來的個人網址）。
    • Email Preferences：輸入 y (同意) 或 n (不同意) 接收通知郵件。
  4. 點擊 「Verify」 完成拼圖或旋轉圖片的人類身份驗證。
  5. 點擊 「Create account」。
  6. 至您的電子信箱收取 8 位數驗證碼，輸入後即可完成註冊！（隨後的問卷調查可直接滑到最下方點擊 Skip personalization 跳過）。
步驟二

個人檔案與雙重驗證 (2FA)

1. 修改個人檔案

點擊右上角個人頭像 → 選擇 「Your profile」 → 點擊 「Edit profile」。您可以在此處上傳頭像、填寫自我介紹（Bio）及社交媒體連結。

2. 開啟雙重驗證 (2FA) 強烈建議

為保護您的帳號不被盜用，請務必開啟 2FA 驗證：

  1. 點擊右上角頭像 → 「Settings」（設定）。
  2. 左側選單選擇 「Password and authentication」。
  3. 在 Two-factor authentication 區塊點擊 「Enable two-factor authentication」。
  4. 使用手機 Authenticator App（如 Google Authenticator）掃描 QR Code 綁定。
  5. ※ 務必下載並妥善保存系統提供的 Recovery codes（恢復金鑰）！
步驟三

開發者必備設定（Git & SSH 金鑰）

1. 安裝 Git 並設定個人資訊

請先於電腦安裝 Git，並在終端機（Terminal / Git Bash）輸入以下指令設定您的身分：

# 設定您的使用者名稱

git config --global user.name "您的GitHub使用者名稱"

# 設定您的電子信箱（需與 GitHub 註冊信箱一致）

git config --global user.email "your_email@example.com"

2. 產生 SSH 金鑰（免密碼推送程式碼）

在終端機中輸入以下指令產生安全金鑰（連續按 Enter 使用預設值即可）：

ssh-keygen -t ed25519 -C "your_email@example.com"

3. 複製金鑰內容

根據您的作業系統，輸入對應指令複製金鑰：

MacOS 用戶
pbcopy < ~/.ssh/id_ed25519.pub
Windows 用戶
clip < ~/.ssh/id_ed25519.pub

4. 將金鑰新增至 GitHub 網站

  1. 回到 GitHub 網站，點擊右上角頭像 → 「Settings」。
  2. 左側選單點擊 「SSH and GPG keys」。
  3. 點擊右上角的 「New SSH key」。
  4. 在 Title 輸入這台電腦的名稱（例如：My Laptop）。
  5. 在 Key 欄位貼上剛剛複製的金鑰內容。
  6. 點擊 「Add SSH key」 按鈕。

5. 測試連線

在終端機輸入以下指令測試是否成功連線：

ssh -T git@github.com

※ 如果看到 Hi [您的帳號]! You've successfully authenticated... 就代表大功告成！

步驟四

建立您的第一個專案倉庫 (Repository)

  1. 回到 GitHub 首頁，點擊左側或右上角的 「New」（新建）按鈕。
  2. 在 Repository name 欄位輸入專案名稱（例如：hello-world）。
  3. 選擇 Public（公開，所有人可見）或 Private（私人，僅限您或指定成員可見）。
  4. 勾選 「Add a README file」（這會自動建立一個說明文件，方便您直接看到效果）。
  5. 點擊最下方的 「Create repository」 按鈕。
恭喜！ 您現在已經擁有一個完整的 GitHub 帳號，並成功建立了第一個程式碼倉庫。您可以開始將您的程式碼推送到雲端，或與全世界的開發者協作了！
