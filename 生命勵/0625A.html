🤖 STEP 1｜建立 LINE Bot Channel LINE Developers
📌 需要一個 LINE 帳號登入 LINE Developers Console，建議使用管理員的 LINE 帳號。

① 進入 LINE Developers Console

1
前往 https://developers.line.biz/
點選右上角「Log in」→ 使用 LINE 帳號登入
2
點選「Create a new provider」
輸入名稱，例如：生命勵活動系統

② 建立 Messaging API Channel

3
在 Provider 頁面點選「Create a new channel」
選擇「Messaging API」
4
填寫以下資訊：
欄位填寫內容
Channel typeMessaging API
Channel name生命勵手搖車活動
Channel description手搖車體驗課程報名系統
Category非營利組織 / 社會福利
Email管理員信箱

③ 取得重要金鑰

5
進入 Channel 後，點選「Messaging API」分頁
滾到最下方找到「Channel access token」
點選「Issue」→ 複製並保存這串 Token
✅ 請將以下資訊記錄下來：

Channel Access Token：

④ 取得管理員群組 ID

6
將 LINE Bot 加入 A 群組
（掃描 Bot 的 QR Code 或搜尋 Bot ID 加入）
7
在群組中傳送任意訊息，透過 Webhook 事件取得 source.groupId
格式範例：C1a2b3c4d5e6f7g8h9i0j1k2l3
💡 快速取得 Group ID 方法：
先設定一個臨時 Webhook URL（如 webhook.site），將 Bot 加入群組後傳訊息，即可在 Webhook 事件中看到 groupId。
✅ 記錄群組 ID：
📱 STEP 2｜建立 LIFF App LINE Developers
📌 LIFF（LINE Front-end Framework）讓網頁可以在 LINE 內開啟並取得使用者資料。

① 新增 LIFF

1
在剛建立的 Channel 頁面，點選上方「LIFF」分頁
點選「Add」新增 LIFF App

② 填寫 LIFF 設定

設定項目填寫值說明
LIFF app name手搖車報名表單顯示名稱
SizeFull全螢幕開啟
Endpoint URL您的表單 HTTPS 網址稍後部署後填入
Scopeprofile chat_message.write必須勾選
Bot link featureOn (Aggressive)自動加好友
Scan QROFF不需要
⚠️ Endpoint URL 必須是 HTTPS，建議先用 https://example.com 佔位，部署後再回來修改。

③ 取得 LIFF ID

3
建立完成後，複製 LIFF ID
格式範例：1234567890-AbCdEfGh
✅ 記錄 LIFF ID：

③ 預設欄位結構（GAS 會自動建立）

欄欄位名稱說明
A編號自動流水號
B報名時間台北時區時間戳記
C姓名報名者真實姓名
D電話聯絡電話
E信箱電子信箱（選填）

💡 不需要手動建立欄位，GAS 程式碼第一次執行時會自動建立標題列並套用樣式。
⚙️ STEP 4｜部署 Google Apps Script GAS
📌 需要建立 兩支 GAS 程式：
  ① 主 GAS（接收表單資料、寫入 Sheets、推播通知）
  ② Webhook GAS（接收 LINE 群組訊息、回覆報名卡片）

① 建立主 GAS（表單接收）

1
開啟剛建立的 Google Sheets
點選上方選單「擴充功能」→「Apps Script」
2
刪除預設的 function myFunction(){}
貼上「GAS 後端 v3」的完整程式碼
3
點選右上角「部署」→「新增部署」
4
設定部署選項：
項目選擇
選取類型網頁應用程式
執行身分我（您的 Google 帳戶）
誰可以存取所有人
5
點選「部署」→ 授權 Google 帳戶存取權限
複製產生的 Web App URL
✅ 記錄主 GAS Web App URL：

② 建立 Webhook GAS（獨立專案）

6
前往 https://script.google.com
點選「新增專案」（不是從 Sheets 開啟）
命名為：LINE Bot Webhook
7
貼上「GAS Webhook v3」的完整程式碼
同樣部署為「網頁應用程式」，存取權限「所有人」
✅ 記錄 Webhook GAS URL：

③ 設定每日統計觸發器

8
在主 GAS 編輯器中，選擇函式 setupTriggers
點選 ▶️ 執行 → 授權後完成
每天早上 09:00 自動推播報名統計摘要
⚠️ 重要：每次修改 GAS 程式碼後，必須重新部署（建立新版本），否則 URL 仍執行舊版本。

🔗 STEP 6｜設定 LINE Bot Webhook LINE Developers

① 設定 Webhook URL

1
回到 LINE Developers Console
進入您的 Messaging API Channel
點選「Messaging API」分頁
2
找到「Webhook settings」區塊
點選「Edit」填入 Webhook GAS URL：
https://script.google.com/macros/s/【Webhook GAS ID】/exec
3
開啟「Use webhook」開關 → 點選「Verify」
出現 Success 表示設定成功

② 關閉自動回覆訊息

4
在「LINE Official Account features」區塊
點選「Auto-reply messages」→ 設為 Disabled
點選「Greeting messages」→ 依需求設定

③ 設定 LIFF Endpoint URL

5
回到「LIFF」分頁
點選您建立的 LIFF App → 「Edit」
將 Endpoint URL 更新為您的表單實際部署網址
