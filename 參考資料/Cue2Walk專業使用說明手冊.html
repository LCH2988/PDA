📋 產品介紹

產品概述

Cue2Walk 是一款專為帕金森氏症患者設計的智能步態訓練輔助系統。透過多模式感官提示（視覺、聽覺、觸覺），有效改善步態凍結、步幅縮短等症狀，提升患者行動能力與生活品質。

步態分析
進度報告

適用對象

  • 確診帕金森氏症患者（Hoehn-Yahr 分期 1-4 期）
  • 出現步態凍結（FOG）症狀者
  • 步幅縮短、行走速度減慢患者
  • 需要步態復健訓練的神經系統疾病患者
  • 有跌倒風險的高齡者
✅ 臨床實證效果：
  • 步幅增加平均 15-25%
  • 行走速度提升 20-30%
  • 步態凍結發生率降低 40-60%
  • 跌倒風險減少 35%

🔧 安裝設定

首次使用設定

1充電準備

首次使用前，請將主機和感測器充電至 100%。充電時指示燈為紅色，充滿後轉為綠色。

2下載應用程式

在智慧型手機或平板電腦上下載「Cue2Walk」應用程式：

  • iOS：App Store 搜尋「Cue2Walk」
  • Android：Google Play 搜尋「Cue2Walk」

3建立帳號

開啟 App 後，點選「註冊」建立個人帳號。需填寫：

  • 姓名、性別、出生日期
  • 診斷資訊（帕金森氏症分期）
  • 身高、體重
  • 主要症狀描述
  • 緊急聯絡人資訊

4配對裝置

開啟主機電源（長按電源鍵 3 秒），在 App 中選擇「新增裝置」：

  1. 確認手機藍牙已開啟
  2. App 會自動搜尋附近裝置
  3. 選擇顯示的「Cue2Walk-XXXX」
  4. 輸入配對碼：0000
  5. 配對成功後會顯示綠色勾勾

5穿戴感測器

將兩個腳踝感測器分別固定於左右腳踝外側：

  • 確保感測器緊貼皮膚（可穿在襪子外）
  • 魔鬼氈鬆緊度以可插入一指為宜
  • 感測器指示燈應朝外側
  • 左腳感測器標示「L」，右腳標示「R」

6校準設定

完成穿戴後，進行初始校準：

  1. 在 App 中選擇「開始校準」
  2. 依照指示站立 10 秒（雙腳與肩同寬）
  3. 原地踏步 20 次
  4. 向前行走 10 步
  5. 系統會自動分析並完成校準
⚠️ 注意事項：
  • 校準時請在平坦地面進行
  • 建議穿著平時慣用的鞋子
  • 若使用助行器，請一併使用進行校準
  • 每次更換鞋子或環境時，建議重新校準

📱 使用方法

日常使用流程

1開機準備

  • 長按主機電源鍵 3 秒開機
  • 確認電量充足（建議 >20%）
  • 穿戴腳踝感測器並確認固定牢靠
  • 將主機夾在腰帶或放入口袋

2啟動訓練模式

在 App 中選擇訓練模式：

  • 自由行走模式：日常活動時使用，提供即時提示
  • 結構化訓練：按照設定的訓練計畫進行
  • 室內模式：在家中練習，投影距離較短
  • 戶外模式：外出使用，增強提示強度

3開始行走

依照視覺或聽覺提示進行行走：

  • 視線注視前方地面的投影線條
  • 配合節拍器節奏邁步
  • 盡量跨過每條線或踩在標記上
  • 保持穩定的步伐節奏

4應對步態凍結

當感測到步態凍結時，系統會：

  • 自動增強視覺提示（閃爍或變色）
  • 發出特殊提示音
  • 震動提醒
  • 語音指導：「抬起腳，跨過線」

解凍技巧：

  1. 停下來，深呼吸
  2. 重心轉移到一側
  3. 想像跨過障礙物
  4. 配合節拍重新起步

5結束與記錄

  • 在 App 中點選「結束訓練」
  • 系統自動儲存訓練數據
  • 查看本次訓練報告
  • 關閉主機電源

特殊情境使用

🚪 通過門檻

  • 提前 2-3 步啟動「門檻模式」
  • 系統會在門檻前投射明顯標記
  • 放慢速度，加大步伐
  • 一次性跨過，避免停頓

🔄 轉彎

  • 轉彎前放慢速度
  • 使用多個小步而非大幅度轉身
  • 系統會調整節拍速度
  • 保持重心穩定

🪜 上下樓梯

⚠️ 注意：上下樓梯時建議關閉地面投影，僅使用聽覺提示，並務必扶好扶手。

🏠 室內狹窄空間

  • 切換至「室內模式」
  • 縮短投影距離
  • 降低步伐速度
  • 增加轉彎頻率設定

訓練環境 多樣化環境（室內外、不同地形）
提示模式 按需提示（僅在困難時啟動）

訓練內容 • 長距離行走（500 公尺以上）

• 購物、外出等實際活動

維持訓練（第 13 週後）

目標：維持訓練成果，融入日常生活
  • 每天使用 Cue2Walk 進行日常活動
  • 每週至少 3 次結構化訓練
  • 定期（每月）進行步態評估
  • 根據評估結果調整訓練計畫

訓練注意事項

  • ⚠️ 訓練前確認藥效時間（建議在藥效最佳時訓練）
  • ⚠️ 感到疲勞或不適時立即休息
  • ⚠️ 初期訓練建議有人陪同
  • ⚠️ 穿著防滑、合腳的鞋子
  • ⚠️ 訓練環境應光線充足、無障礙物
  • ⚠️ 定期與醫療團隊討論訓練進度

⚠️ 安全須知

使用前必讀

🚫 禁忌症（請勿使用）：
  • 嚴重心血管疾病未控制者
  • 急性期骨折或關節損傷
  • 嚴重認知功能障礙（MMSE < 18）
  • 癲癇發作未控制者（光敏感性癲癇）
  • 嚴重視力障礙無法看見投影
  • 嚴重聽力障礙無法聽見提示音
⚠️ 謹慎使用（需醫師評估）：
  • 姿勢性低血壓頻繁發作
  • 近期有跌倒病史（3 個月內）
  • 使用助行器或輪椅者
  • 合併其他神經系統疾病
  • 嚴重關節炎影響行走

使用安全指引

🏠 環境安全

  • ✓ 確保訓練區域無障礙物、雜物
  • ✓ 地面保持乾燥、無濕滑
  • ✓ 光線充足但避免強光直射
  • ✓ 移除地毯、電線等絆倒危險
  • ✓ 準備穩固的椅子以便隨時休息

👤 個人安全

  • ✓ 初次使用建議有人陪同
  • ✓ 攜帶手機以便緊急聯絡
  • ✓ 穿著合適的防滑鞋
  • ✓ 避免在藥效低落時訓練
  • ✓ 感到頭暈、胸悶立即停止

⚡ 裝置安全

  • ✓ 使用原廠充電器和電池
  • ✓ 避免在極端溫度環境使用
  • ✓ 不可浸泡在水中
  • ✓ 發現異常發熱立即停用
  • ✓ 定期檢查穿戴部件是否損壞

緊急情況處理

🚨 跌倒處理

  1. 保持冷靜，不要急於起身
  2. 檢查是否有疼痛或受傷
  3. 若無大礙，側躺後慢慢起身
  4. 使用穩固支撐物協助站立
  5. 若無法起身或疑似骨折，立即求助
  6. 系統會自動偵測跌倒並發送警報給緊急聯絡人

🩺 身體不適處理

出現以下症狀請立即停止使用：
  • 胸痛、胸悶、呼吸困難
  • 嚴重頭暈、視力模糊
  • 心悸、冷汗
  • 嚴重疲勞、無力
  • 關節或肌肉劇烈疼痛

處理步驟：

  1. 立即坐下或躺下休息
  2. 按下裝置緊急按鈕
  3. 聯絡家人或緊急聯絡人
  4. 必要時撥打 119

兒童與寵物安全

  • ⚠️ 將裝置放置於兒童無法觸及處
  • ⚠️ 小零件可能造成窒息危險
  • ⚠️ 充電時避免兒童接近
  • ⚠️ 投影光線避免直射眼睛
  • ⚠️ 寵物可能被投影光線吸引，訓練時注意

🔧 維護保養

日常清潔

主機清潔

  • 每次使用後用乾布擦拭外殼
  • 投影鏡頭用專用鏡頭布輕輕擦拭
  • 避免使用酒精或化學溶劑
  • 不可用水直接沖洗

感測器清潔

  • 每週用微濕布擦拭感測器表面
  • 魔鬼氈黏附毛髮時用小刷子清理
  • 綁帶可拆卸手洗（晾乾後再使用）
  • 確保感測器接點無灰塵

電池維護

延長電池壽命的方法：
  • ✓ 避免完全放電（保持 20% 以上）
  • ✓ 長期不使用時，每月充電一次
  • ✓ 充電時避免高溫環境
  • ✓ 使用原廠充電器
  • ✓ 電池壽命約 500 次充電循環（約 2 年）

定期檢查

檢查項目 頻率 檢查內容
外觀檢查 每週 檢查是否有裂痕、損壞
功能測試 每週 測試投影、聲音、震動功能
綁帶檢查 每週 檢查魔鬼氈黏性、綁帶彈性
軟體更新 每月 檢查並安裝最新版本
校準 每月 重新進行系統校準
專業檢測 每年 送回原廠或授權中心檢測

儲存方式

長期不使用時（超過 1 個月）：
  • 充電至 50-60%
  • 關閉電源
  • 放入收納包
  • 存放於乾燥、陰涼處
  • 避免陽光直射
  • 溫度 15-25°C，濕度 < 60%

軟體更新

更新步驟

  1. 確保裝置電量 > 50%
  2. 連接 Wi-Fi
  3. 開啟 App，進入「設定」→「系統更新」
  4. 點選「檢查更新」
  5. 若有新版本，點選「下載並安裝」
  6. 更新過程約 5-10 分鐘，請勿關閉裝置
  7. 更新完成後會自動重啟
⚠️ 更新注意事項：
  • 更新前建議備份訓練數據
  • 更新期間不可使用裝置
  • 更新失敗請聯絡客服

🔍 故障排除

常見問題與解決方法

❌ 問題：裝置無法開機

可能原因與解決方法：
  • ✓ 電池耗盡 → 充電 30 分鐘後再試
  • ✓ 電源鍵未長按 → 長按 3 秒以上
  • ✓ 系統當機 → 同時按住電源鍵和音量鍵 10 秒強制重啟
  • ✓ 硬體故障 → 聯絡客服維修

❌ 問題：無法連接藍牙

解決步驟：
  1. 確認手機藍牙已開啟
  2. 確認裝置電源已開啟
  3. 在手機藍牙設定中「忽略此裝置」
  4. 重新啟動手機和裝置
  5. 重新配對
  6. 若仍無法連接，刪除 App 重新安裝

❌ 問題：投影不清晰或無投影

檢查項目：
  • ✓ 鏡頭是否髒污 → 用鏡頭布清潔
  • ✓ 投影距離是否過遠 → 調整至 0.5-2.5 公尺
  • ✓ 環境光線過強 → 調高投影亮度或改善照明
  • ✓ 地面顏色過淺 → 投影在深色地面效果較佳
  • ✓ 投影功能是否開啟 → 檢查 App 設定

❌ 問題：聽不到提示音

檢查項目：
  • ✓ 音量是否過小 → 調高音量
  • ✓ 是否開啟靜音 → 取消靜音模式
  • ✓ 喇叭孔是否堵塞 → 清理喇叭孔
  • ✓ 聽覺提示是否啟用 → 檢查 App 設定

❌ 問題：感測器數據不準確

解決方法：
  • ✓ 重新校準系統
  • ✓ 確認感測器穿戴位置正確
  • ✓ 檢查綁帶鬆緊度（不可過鬆或過緊）
  • ✓ 確認感測器電量充足
  • ✓ 更新至最新軟體版本

❌ 問題：電池續航力下降

可能原因：
  • ✓ 電池老化（使用 2 年以上）→ 聯絡客服更換電池
  • ✓ 投影亮度過高 → 適度降低亮度
  • ✓ 音量過大 → 調整至適中音量
  • ✓ 背景 App 過多 → 關閉不必要的功能
  • ✓ 環境溫度過低 → 避免在低溫環境使用

❌ 問題：App 閃退或當機

解決步驟：
  1. 強制關閉 App 後重新開啟
  2. 重新啟動手機
  3. 檢查 App 是否為最新版本
  4. 清除 App 快取資料
  5. 刪除 App 重新安裝（注意備份數據）
  6. 確認手機系統版本符合需求

❌ 問題：訓練數據未同步

解決方法：
  • ✓ 確認網路連線正常
  • ✓ 檢查帳號登入狀態
  • ✓ 手動點選「同步數據」
  • ✓ 確認雲端儲存空間充足
  • ✓ 重新登入帳號
